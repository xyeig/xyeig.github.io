Hexo + github 搭建个人博客

发表于 2017-06-09

1 安装Hexo

在安装前，您必须检查电脑中是否已安装下列应用程序：

  • Git
  • Node.js

npm install -g hexo-cli
在电脑的任意盘新建文件夹 yourgithubname.github.io
在该文件夹下执行
hexo init
npm install
hexo generate / hexo g
hexo server / hexo s

2 安装Hexo next主题

git clone https://github.com/iissnan/hexo-theme-next
修改主题配置文件 _config.yml
scheme: Pisces

3 配置SSH 将本地博客同步到github

github上新建仓库，命名为yourgithubname.github.io
安装deployer插件
npm install hexo-deployer-git –save

修改 _config.yml
deploy:
type: git
repository: https://github.com/xyeig/xyeig.github.io
branch: master
执行
hexo clean / hexo clean
hexo generate / hexo g
hexo deploy / hexo d
